Lab 5 - JSPs

Welcome to a servlet-less webapp!

Servlets are java code that surrounds HTML (and other things). JSPs are "inside-out" servlets, that is, they are HTML that can have java code embedded in it.

This page is an html page (as opposed to other labs), our lab JSP is called example.jsp
